Programme d'Entraînement

Dos & Biceps

Tractions (ou tirage vertical)

3 séries 8-10 répétitions

Technique : Saisissez la barre, mains écartées. Tirez votre corps vers le haut en contractant le dos jusqu'à ce que votre menton dépasse la barre.

Repos : 90 secondes entre les séries

Rowing avec haltères

3 séries 10-12 répétitions

Technique : Penchez-vous en avant, dos droit. Tirez l'haltère vers votre hanche en contractant le dos.

Repos : 60-90 secondes entre les séries

Curl biceps avec haltères

3 séries 10-12 répétitions

Technique : Debout, bras le long du corps. Fléchissez les coudes pour amener les haltères vers vos épaules.

Repos : 60 secondes entre les séries

Curl marteau

3 séries 10-12 répétitions

Technique : Comme le curl biceps mais avec les paumes face à face pendant tout le mouvement.

Repos : 60 secondes entre les séries

Jour de repos

Récupération active ou légère cardio si souhaité

Jambes

Squats

3 séries 10-12 répétitions

Technique : Pieds écartés largeur d'épaules, descendez comme pour vous asseoir, en gardant le dos droit et les genoux alignés avec les pieds.

Repos : 90 secondes entre les séries

Fentes avant

3 séries 10 répétitions par jambe

Technique : Faites un grand pas en avant, fléchissez les genoux pour descendre, puis revenez à la position de départ.

Repos : 60-90 secondes entre les séries

Extensions de jambes

3 séries 12-15 répétitions

Technique : Assis sur la machine, étendez les jambes devant vous en contractant les quadriceps.

Repos : 60 secondes entre les séries

Mollets debout

3 séries 15-20 répétitions

Technique : Debout sur le bord d'une marche, montez sur la pointe des pieds puis redescendez.

Repos : 45-60 secondes entre les séries

Pectoraux & Triceps

Développé couché

3 séries 8-10 répétitions

Technique : Allongé sur un banc, poussez la barre vers le haut en contractant les pectoraux, puis redescendez lentement.

Repos : 90 secondes entre les séries

Pompes

3 séries Max répétitions

Technique : En position de planche, descendez en fléchissant les coudes puis remontez en les tendant.

Repos : 60-90 secondes entre les séries

Extensions triceps à la poulie

3 séries 10-12 répétitions

Technique : Debout face à la poulie haute, tirez la corde vers le bas en tendant les bras.

Repos : 60 secondes entre les séries

Dips

3 séries 8-10 répétitions

Technique : En appui sur les barres parallèles, fléchissez les coudes pour descendre puis remontez.

Repos : 90 secondes entre les séries

Jour de repos

Récupération active ou légère cardio si souhaité

Full Body

Squat avec poids

3 séries 10 répétitions

Technique : Squat classique avec haltères ou barre sur les épaules.

Repos : 90 secondes entre les séries

Développé épaules

3 séries 10 répétitions

Technique : Assis ou debout, poussez les haltères au-dessus de votre tête.

Repos : 60-90 secondes entre les séries

Rowing à un bras

3 séries 10 répétitions par bras

Technique : Un genou et une main sur un banc, tirez l'haltère vers votre hanche.

Repos : 60 secondes entre les séries

Planche abdominale

3 séries 30-45 secondes

Technique : En appui sur les avant-bras et les orteils, maintenez votre corps en ligne droite.

Repos : 45-60 secondes entre les séries

Progression Hebdomadaire

Conseils Essentiels

Sommeil

Dors 8-9 heures par nuit. C'est pendant le sommeil que ton corps libère l'hormone de croissance et répare tes muscles.

Hydratation

Bois au moins 2,5-3L d'eau par jour. Une déshydratation de seulement 2% peut réduire tes performances de 20%.

Respiration

Expire pendant l'effort (phase concentrique) et inspire pendant le retour (phase excentrique). Ne bloque jamais ta respiration.

Constance

La régularité est plus importante que l'intensité. Mieux vaut 3-4 séances modérées par semaine que 2 séances intenses suivies d'abandon.

Conseils Supplémentaires

Alimentation

Mange toutes les 3-4 heures pour maintenir un apport constant de nutriments. Privilégie les aliments complets et non transformés.

Récupération

Respecte 48h de repos entre deux entraînements du même groupe musculaire. La croissance se fait pendant la récupération, pas pendant l'effort.

Progression

Augmente progressivement les charges ou les répétitions. Ton corps s'adapte, il faut constamment le stimuler avec de nouveaux défis.

Note importante

À ton âge, privilégie la technique parfaite plutôt que les charges lourdes. Ton corps est encore en développement, il est crucial d'éviter les blessures qui pourraient compromettre ta croissance naturelle.

Conseil du Jour

Aujourd'hui, concentre-toi sur la qualité de tes mouvements plutôt que sur le poids. Ralentis la phase excentrique (descente) pour maximiser la tension musculaire.
